# webdev-final-project

### By Taite Dodson

## About

This is my final project for CS 563 at Portland Statue University. This acts as a pseudo resume. Going over my accomplishments in my academic and professional life.

## How to run

There are no plugins or special tools needed to run this code, it's a simple HTML/Javascript/CSS website. If this code were to be copied the main thing that would need to change would be the formspree link as that is set up specifically for my account. The website itself is hosted on Github pages under [https://dodsonta.github.io/webdev-final-project/](https://dodsonta.github.io/webdev-final-project/)
